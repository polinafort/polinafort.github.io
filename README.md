# Галерея раскрасок для взрослых

Простой сайт для скачивания и распечатки раскрасок.

## Функции
- Просмотр раскрасок
- Скачивание PNG-файлов
- Адаптивный дизайн

## Как использовать
1. Перейдите на сайт
2. Выберите раскраску
3. Нажмите "Скачать PNG"
4. Распечатайте и раскрашивайте!

## Технологии
- HTML5
- CSS3
- JavaScript
